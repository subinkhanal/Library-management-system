About Library Management System




Before discussing library management systems, it might help to talk about systems. A system is something formed of parts, each of which interacts with the other parts to achieve some common purpose. In the case of a library management system, the parts work together to support the management of library information resources: their acquisition, representation and circulation. Each of these parts constitutes a subsystem which in turn comprises a set of interconnected parts.
